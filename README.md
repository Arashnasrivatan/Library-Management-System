## Features

- Signup 💎
- Login 💎
- Reset password 💎
- Reserve book 💎
- See Reserved Books 💎
- Add Book by User 💎
- Edit Profile 💎
- Logout 💎
- Search 💎
## Admin panel Features
- Accept or Reject Books 💎
- Show/Delete Books 💎
- See User details 💎
- Add Admin 💎
- Edit Admin Username and Password 💎
- Logout From Admin panel 💎
